DESIGN PATTERNS USED
====================
1. Singleton Pattern        - AccountRepository
2. Repository Pattern       - Data access abstraction
3. Service Layer Pattern    - Business logic separation
4. MVC Pattern             - Overall architecture

KEY TECHNOLOGIES
================
- Java 11+ (no external dependencies)
- Object-Oriented Programming
- Exception Handling
- Collections Framework (HashMap, ArrayList)
- BigDecimal for financial calculations
- LocalDateTime for timestamps
- Enums for type safety

FEATURES IMPLEMENTED
====================
1. User Authentication      - User ID + PIN with security
2. Cash Withdrawal         - With limits and validations
3. Cash Deposit           - With maximum limits
4. Money Transfer         - Atomic transactions
5. Transaction History    - Complete audit trail
6. Account Summary        - Detailed information
7. PIN Change            - With strength validation
